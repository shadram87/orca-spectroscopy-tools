# Contributing

Bug reports, test data, and focused pull requests are welcome.

1. Create a branch from `main`.
2. Install the development environment with `pip install -e ".[dev]"`.
3. Add or update tests for behavioral changes.
4. Run `pytest` and `ruff check .` before opening a pull request.

Please do not commit proprietary ORCA binaries, licensed software, or large raw calculation files.
Small synthetic or openly shareable text fixtures are preferred for tests and examples.
